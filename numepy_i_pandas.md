Основные функции и концепции NumPy [(на базу)](база.md):

1. Создание массивов:
```
import numpy as np
arr = np.array([1, 2, 3])           # Создание из списка
zeros = np.zeros((3, 3))            # Массив из нулей
ones = np.ones((2, 2))              # Массив из единиц
arange = np.arange(0, 10, 2)        # Диапазон с шагом
linspace = np.linspace(0, 1, 5)     # Равномерное разбиение
```
2. Атрибуты массива:
```
arr.shape                           # Размерность массива
arr.ndim                            # Количество осей (измерений)
arr.size                            # Общее количество элементов
arr.dtype                           # Тип данных элементов
```
3. Операции с массивами:
```
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
c = a + b                           # Поэлементное сложение
d = a * b                           # Поэлементное умножение
e = np.dot(a, b)                    # Скалярное произведение
f = a ** 2                          # Возведение в степень
```
4. Индексация и срезы:
```
arr = np.array([[1, 2, 3], [4, 5, 6]])
item = arr[0, 1]                    # Элемент на позиции (0, 1)
row = arr[1, :]                     # Вся вторая строка
column = arr[:, 2]                  # Весь третий столбец
slice = arr[0:2, 1:3]               # Срез массива
```

5. Форма и преобразования:
```
arr.reshape((3, 2))                 # Изменение формы массива
arr.flatten()                       # Преобразование в одномерный массив
arr.transpose()                     # Транспонирование матрицы
```

6. Агрегатные функции:
```
np.sum(arr)                         # Сумма всех элементов
np.mean(arr)                        # Среднее значение
np.min(arr)                         # Минимальное значение
np.max(arr)                         # Максимальное значение
np.std(arr)                         # Стандартное отклонение
```

7. Логические операции:
```
mask = arr > 2                      # Булева маска
filtered = arr[mask]                # Фильтрация по условию
np.any(mask)                        # Проверка хотя бы одного True
np.all(mask)                        # Проверка всех True
```

8. Генерация случайных чисел:
```
random_arr = np.random.rand(3, 3)   # Равномерное распределение [0, 1)
normal_arr = np.random.randn(3, 3)  # Нормальное распределение
int_arr = np.random.randint(0, 10, (3, 3)) # Целые числа в диапазоне
```

9. Математические функции:
```
np.exp(arr)                         # Экспонента
np.log(arr)                         # Натуральный логарифм
np.sqrt(arr)                        # Квадратный корень
np.sin(arr)                         # Синус
```

10. Загрузка и сохранение данных:
```
np.save('file.npy', arr)            # Сохранение массива в файл
loaded = np.load('file.npy')        # Загрузка массива из файла
np.savetxt('file.txt', arr)         # Сохранение в текстовый файл
text_loaded = np.loadtxt('file.txt') # Загрузка из текстового файла
